Tentang Saya

Saya mulai tertarik dengan dunia teknologi sejak duduk di bangku SMA. Ketertarikan tersebut terus berkembang hingga saat ini saya menempuh pendidikan di Universitas UPN Veteran Jakarta. Saya memiliki minat khusus di bidang web development, terutama pada bagian frontend, karena saya menikmati proses membangun tampilan yang menarik sekaligus memberikan pengalaman pengguna yang baik. Selain itu, saya adalah pribadi yang senang belajar hal baru dan terus mengembangkan kemampuan, baik dari sisi teknis maupun kreativitas dalam membuat sebuah aplikasi. Saya percaya bahwa teknologi dapat menjadi solusi untuk berbagai permasalahan, dan saya ingin menjadi bagian dari perkembangan tersebut.
